# PDF 解析与证据定位

## 两层模型

`ParsedDocument` 保存客观版面事实；`PaperDeconstruction` 保存科研语义判断。两者通过稳定 artifact/evidence ID 关联。

## 权利闸门

只有以下来源之一经过确认，结构化结果才可持久化：

- 开放全文。
- 用户拥有的私有副本。
- 机构授权访问。

PDF 原文件默认不入数据库；保存文件哈希、解析器版本和结构化结果即可。

## 解析器

- PyMuPDF：本地基线，当前已有可运行适配器。
- GROBID：保留 TEI 适配接口，待 R2 接服务。
- MinerU：保留 JSON 适配接口，待 R2 完成字段映射。

扫描版、跨页表格、复杂双栏和图注断行必须进入人工复核，不能静默当作正确结果。
